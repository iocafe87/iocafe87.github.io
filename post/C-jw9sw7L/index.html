在Mac下设置ll、la等命令别名

2021-06-09 1 min read # macOS # alias

Linux下的ll，la用起来非常方便，但是Mac下默认是没有这些别名的。我们可以自己配置一下别名。把指令配置到~/.bash_profile文件当中即可（如果没有，请自行创建）。

vim ~/.bash_profile

添加这些别名（也可根据自己的喜好添加参数）：

alias ll='ls -l'
alias la='ls -Al'

保存文件之后，执行一下source ~/.bash_profile，即可生效了。
